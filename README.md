# KDE-Overview-on-lock-screen
A script that opens the kde overview and screenshots it before then locking the Computer

<img width="2560" height="1440" alt="image" src="https://github.com/user-attachments/assets/c3260ac8-0d23-4ab6-b4a0-300917d84cf4" />
To use this script, run it for the first time to create the image, and then add the image as a lock screen wallpaper which should be located in your home folder.
